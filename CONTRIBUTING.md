# Contributing

Small, focused improvements are welcome.

1. Fork the repository.
2. Create a branch: `git switch -c feature/your-change`.
3. Test the page at desktop and mobile widths.
4. Commit with a clear message.
5. Open a pull request explaining the visual or functional change.

Please keep RICEPRINT dependency-free unless a dependency solves a substantial problem that cannot reasonably be handled in the browser.
